To create your first connection, use the **Connections** Menu and click on Add New Connection

![](screenshots/Connection%20Configuration%20Dialog.png)

When first connection is being saved, you will be asked to provide the encryption key. 

This key is then used to encrypt the connection details that will be stored on your local computer.

![](screenshots/Encryption%20Key%20Dialog.png)

Once the connection data is encrypted, program will ask for the decryption key on every startup.

![](screenshots/Decryption%20Key%20Dialog.png)

If you forget the encryption key, or provide incorrect one, program will not restore any session information, so you will need to provide new session details to overwrite previous configuration or restart program to enter the encryption key again. 
If you provided the right encryption key on the startup, you will see your connection available in the menu.

![](screenshots/Edit%20Connection%20Menu.png)

From here you can connect, edit or delete connection info from the registry.

Deleting all connections will return the configuration to the point where adding any new connection will result in setting new encryption key.
